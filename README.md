<h1 align="center">Projet CAFR </h1>
<p>
  Le projet de Chat d'Aide et Fiable a la Revision (CAFR), me challenge a la mise en place d'un chatbot IA Retrieval-Augmented Generation, hébergé sur une machine virtuelle en local.
  Le Chat devra apporté des réponse avec pour unique source des document des cours Unilasalle.  
</p>
Les Étapes identifiées
<ul>
  <li>Instalation de la machine virtuelle</li>
  <li>Instalation des dépendances</li>
  <li>Intégration des documents</li>
  <li>Parametrage du RAG</li>
</ul>

Ce projet est ma premiere experience solo avec l'IA, j'ai donc utilisé l'asistance de Claude pour sa réalisation.

Deux solution se presente a moi pour l'hébergement de la machine:
Virtual box ou VMware. Selon toute vraisemblance Virtualbox reste un logiciel Open source et performant pour une utilisation non intensive, je reste donc sur cette option.
J'intalle une machine Linux Ubuntu Server 24.04.3.Ce choix est justifié par l'open source et le fait que je dois ammelioré mes compétences dans la manipulation en tout genre d'un systeme linux (Un prochain petit projet pourrait etre de passé mon ordinateur de jeu en dual-boot avec un OS linux corespondant a mes besoin, et un OS Windows).
Je crée donc une machine nommé LINUX-CARF
